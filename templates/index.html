Text decoder

This web page is served through a very simple implementation of an HTTP server with Go. It is made to explore the communication between Go and a HTML page while resolving a very simple problem.

The problem is the next one:
You got a ciphered message. To decipher it, you must follow the next rules:

  1. Use the first appearance of the 26 lowercased letters of the English alphabet in the key as the order for the substitution table.
  2. Align the substitution table with the regular English alphabet.
  3. Replace every letter in the ciphered message using the resultant table.
  4. Spaces are kept and the resultant message is the decoded message in lower case.

Example:
Key: The quick brown fox jumps over the lazy dog
Encoded Message: vkbs bs t suepuv
Decoded Message: this is a secret
